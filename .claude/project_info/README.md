# Project Information Directory

This directory contains all project documentation and specifications for the Price Comparison POC.

## Directory Structure

- `infrastructure/` - Infrastructure and deployment specifications
- `design_ui_ux/` - UI/UX designs, mockups, and style guides
- `user_stories/` - User stories and requirements
- `todo/` - Task management and development todos
- `roadmap/` - Project roadmap and milestones
- `pdfs/` - PDF documentation and references
- `business_rules/` - Business logic and validation rules
- `api_specs/` - API documentation and specifications
